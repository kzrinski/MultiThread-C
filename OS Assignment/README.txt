OS ASSIGNMENT BY KYLE ZRINSKI

- All .c files are located in /source
- All .h files are located in /include
- Example task_files and bash script to randomly generate a valid task_file
  is included in resource
- All .o files WILL be in /build once the program has been compiled

TO COMPILE:
1) run make clean
2) run make

TO COMPILE WITH TERMINAL LOGS (DEBUG):
1) run make clean
2) run make debug

TO RUN PROGRAM:
- via the terminal call the executable in the following format once
  compiled: "./scheduler DIRECTORY/TASKFILE QUEUE_SIZE" any additional
  inputs will not be accepted nor will missing one.
